# TeamoRouter Blog

The official blog for **[TeamoRouter](https://router.teamolab.com)** — the native AI LLM router built for OpenClaw.

## What is TeamoRouter?

TeamoRouter gives your OpenClaw agent one API key to access every top LLM (Claude, GPT, Gemini, DeepSeek, Kimi, MiniMax) at up to 50% off official prices. Smart routing automatically picks the best model for each request.

**Install in 2 seconds:** Just paste this into your OpenClaw:
```
Read https://gateway.teamo.ai/skill.md and follow the instructions to install TeamoRouter.
```
